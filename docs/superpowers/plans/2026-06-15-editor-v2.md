# Document Editor v2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Turn the v1 Plate "Write" tab into a full document editor: draft autosave + explicit publish, a pinned layout, a comprehensive toolbar with 15 embeddable fonts, an image uploader, an exact paginated preview with a page navigator, and correct Owner+admin signing.

**Architecture:** Build on branch `feat/plate-html-editor` and the v1 custom `valueToHtml` serializer. Split the one expensive save into cheap DB-only draft autosave plus a deliberate DocuSeal publish (tracked via a new `published_html` snapshot column). Expand the serializer + DocuSeal output shell in lockstep with each new toolbar feature, gated by round-trip tests. Keep the editor a continuous writing surface; pagination lives in a Paged.js preview and a page rail.

**Tech Stack:** Next.js 16 App Router, TypeScript strict, Supabase (Postgres + Storage), Plate v53 (`platejs`, `@platejs/*`), Paged.js, Tailwind v4 + CSS Modules, Phosphor Icons, motion/react.

**Reference:** design doc `docs/plans/2026-06-15-editor-v2-design.md`.

## Critical conventions (read first)

- **Verify Plate v53 APIs against installed source before writing UI** that uses a new plugin. v1 proved the published docs lag the installed version. For any new package, after install run:
  ```bash
  node -e "const r=require('@platejs/<pkg>/react'); console.log(Object.keys(r).filter(k=>/Plugin$/.test(k)).join(', '))"
  ```
  and for transforms: `node -e "const {createSlateEditor}=require('platejs'); ... console.log(Object.keys(editor.tf))"`. Drive serializer node-type strings off the live `KEYS` export (`node -e "console.log(require('platejs').KEYS)"`), never guess.
- **Every serializer change ships with a round-trip test case** in `__tests__/html-serialize.test.ts`. The pattern: build a Value in the normalized editor shape, assert `valueToHtml(value)` equals expected HTML, and assert `valueToHtml(deserialize(valueToHtml(value))) === valueToHtml(value)`. A wrong type string silently drops content from a signed legal document.
- **Run from `apps/web/`.** Type check: `pnpm exec tsc --noEmit`. Test: `pnpm exec vitest run <file>`. Lint: `pnpm exec eslint <files>`.
- **No `transition-all`, no default Tailwind colors, brand tokens only** (`var(--color-brand)` etc.). Marks/blocks that carry styling serialize to inline `style="…"`; never to class names DocuSeal will not have.
- **Dev smoke flow:** dev auth only works on port 4000. Log in at `http://localhost:4000/api/dev/auth`, then a dev server on any free port shares the `localhost` cookie. WebGL is off in gstack browse but the editor is DOM-based, so gstack browse works for it.

```typescript
"use server";

import { revalidatePath } from "next/cache";
import { createClient } from "@/lib/supabase/server";
import { createTemplateFromHtml } from "@/lib/signing/docuseal";
import {
  getDocumentTemplate,
  updateDocumentTemplateRecord,
} from "@/lib/admin/document-templates";
import { wrapInDocumentShell } from "./document-shell";

async function requireAdmin(): Promise<{ error: string | null }> {
  const supabase = await createClient();
  const { data: { user } } = await supabase.auth.getUser();
  if (!user) return { error: "You must be signed in." };
  const { data: profile } = await supabase
    .from("profiles").select("role").eq("id", user.id).maybeSingle();
  if ((profile as { role: string } | null)?.role !== "admin")
    return { error: "Admin access required." };
  return { error: null };
}

/** Cheap, DocuSeal-free draft persistence. Called by autosave + tab switch. */
export async function saveTemplateDraftAction(
  templateId: string,
  html: string,
): Promise<{ ok: true } | { ok: false; error: string }> {
  const { error } = await requireAdmin();
  if (error) return { ok: false, error };
  if (!templateId?.trim()) return { ok: false, error: "Invalid template ID." };
  const template = await getDocumentTemplate(templateId);
  if (!template) return { ok: false, error: "Template not found." };
  if (template.source_html === null)
    return { ok: false, error: "This template is not an HTML document." };
  const ok = await updateDocumentTemplateRecord(templateId, { source_html: html });
  if (!ok) return { ok: false, error: "Could not save your draft. Try again." };
  revalidatePath(`/admin/paperwork/templates/${templateId}`);
  return { ok: true };
}

/** Deliberate: (re)build the DocuSeal template from the current draft, passing
 *  signer roles as parties, and snapshot what was published. Resets fields. */
export async function publishTemplateAction(
  templateId: string,
): Promise<{ ok: true; newDocusealId: number } | { ok: false; error: string }> {
  const { error } = await requireAdmin();
  if (error) return { ok: false, error };
  const template = await getDocumentTemplate(templateId);
  if (!template) return { ok: false, error: "Template not found." };
  if (template.source_html === null)
    return { ok: false, error: "This template is not an HTML document." };

  const fullHtml = wrapInDocumentShell(template.source_html);
  const result = await createTemplateFromHtml(template.display_name, fullHtml, {
    // Parties so the Fields builder shows Owner AND the admin (Proxy).
    submitters: template.signer_roles.map((name) => ({ name })),
  });
  if (!result)
    return { ok: false, error: "Could not publish to DocuSeal. Check DOCUSEAL_API_TOKEN." };

  const ok = await updateDocumentTemplateRecord(templateId, {
    docuseal_template_id: result.templateId,
    published_html: template.source_html,
    is_active: false,
  });
  if (!ok) return { ok: false, error: "Published but the DB update failed. Try again." };

  revalidatePath("/admin/paperwork/templates");
  revalidatePath(`/admin/paperwork/templates/${templateId}`);
  return { ok: true, newDocusealId: result.templateId };
}
```

## Task 3: Autosave hook

**Files:**
- Create: `apps/web/src/app/(admin)/admin/paperwork/templates/[id]/editor/useAutosave.ts`

- [ ] **Step 1: Implement the debounced autosave**

```typescript
"use client";

import { useCallback, useEffect, useRef, useState } from "react";

export type SaveState = "idle" | "saving" | "saved" | "error";

/**
 * Debounced autosave. `serialize` returns the current HTML; `save` persists it.
 * Call `markDirty()` on real edits. `flush()` saves immediately (tab switch /
 * unload). Tracks lastSavedHtml so identical content is not re-sent.
 */
export function useAutosave({
  serialize,
  save,
  delay = 1500,
}: {
  serialize: () => string;
  save: (html: string) => Promise<{ ok: boolean }>;
  delay?: number;
}) {
  const [state, setState] = useState<SaveState>("idle");
  const lastSaved = useRef<string | null>(null);
  const timer = useRef<ReturnType<typeof setTimeout> | null>(null);
  const inFlight = useRef(false);

  const doSave = useCallback(async () => {
    const html = serialize();
    if (html === lastSaved.current) return;
    if (inFlight.current) return;
    inFlight.current = true;
    setState("saving");
    const res = await save(html);
    inFlight.current = false;
    if (res.ok) {
      lastSaved.current = html;
      setState("saved");
    } else {
      setState("error");
    }
  }, [serialize, save]);

  const markDirty = useCallback(() => {
    if (timer.current) clearTimeout(timer.current);
    timer.current = setTimeout(() => void doSave(), delay);
  }, [doSave, delay]);

  const flush = useCallback(async () => {
    if (timer.current) clearTimeout(timer.current);
    await doSave();
  }, [doSave]);

  // Mark the current content as the saved baseline (call once after initial load).
  const setBaseline = useCallback((html: string) => {
    lastSaved.current = html;
    setState("saved");
  }, []);

  useEffect(() => () => { if (timer.current) clearTimeout(timer.current); }, []);

  return { state, markDirty, flush, setBaseline };
}
```
